Topic One: Deep Learning for Object Detection and Tracking

There are two kinds of classical tracking method. The one is generative model focusing on the description of object itself such as sparse coding, online density estimation and PCA. Another one is discriminative model which is called tracking-by-detection such as multiple instance learning and structured SVM. Discriminative model become more dominant in recent several years. Although deep learning has achieved great success in many fields, it doesnâ€™t behave well in object tracking especially multi-target tracking because of several difficulties.
